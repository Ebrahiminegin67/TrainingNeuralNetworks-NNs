# Training Neural Networks

In this project, we learn how to create and train a **basic neural network model** using **PyTorch**.  
We build a **classification model** on the **Fashion-MNIST** dataset and compare the performance of **Fully Connected Networks (FCN)** with different design choices.

---

##  Key Topics
- Basic PyTorch operations  
- Training a deep neural network  
- Improving performance with different **optimizers** and **learning rate schedulers**

---

##  Tools & Libraries
- **Python**  
- **PyTorch**  
- **torchvision**  
- **NumPy**, **Pandas**  
- **Matplotlib**, **scikit-learn**
